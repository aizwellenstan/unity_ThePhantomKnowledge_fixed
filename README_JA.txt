The Phantom KnowledgeプロジェクトデータはUCLの元に公開されています。2017.1.1f1以降のUnityで実行可能です。
Windows Direct X11、Mac Metal 環境に対応しています。それぞれ、推奨動作環境は次の通りです。

Windows: CPU 3.5Ghz(4コア)以上 RAM 16GB Geforce 960GTX 以上
Mac:     CPU 3.5Ghz(4コア)以上 RAM 16GB Radeon R9 M395 以上

もしプロジェクトの再生が重い場合はInspectorウインドウとTimelineウインドウを閉じてから再生してみてください
